# PrajaPulse-civic-accountability-platform
PrajaPulse (ಪ್ರಜಾ ಪಲ್ಸ್) is a civic accountability platform for Bangalore residents to report local issues — water, roads, power, garbage — on a live geotagged map, track complaints by ward, and hold MLAs accountable through promise scorecards. Powered by AI-based multilingual complaint classification using the Claude API.
